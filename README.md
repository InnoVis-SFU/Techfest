# Tech Fest Website

Static rebuild of [codesignexplore.com](https://www.codesignexplore.com/), hosted free on GitHub Pages.

## Live site

After deployment: **https://innovis-sfu.github.io/Techfest/**

## Local preview

Open `index.html` in a browser, or run a local server:

```powershell
cd "c:\Users\shahb\OneDrive\Desktop\Tectfest"
python -m http.server 8080
```

Then visit http://localhost:8080

## Deploy to GitHub Pages

1. Initialize git and push to the repo:

```powershell
cd "c:\Users\shahb\OneDrive\Desktop\Tectfest"
git init
git add .
git commit -m "Add static Tech Fest site migrated from Wix"
git branch -M main
git remote add origin https://github.com/InnoVis-SFU/Techfest.git
git push -u origin main
```

2. On GitHub: **Settings → Pages → Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main** / **/ (root)**

3. Wait 1–2 minutes. The site will be live at `https://innovis-sfu.github.io/Techfest/`.

## Custom domain (optional)

To keep using `codesignexplore.com` without Wix:

1. In repo **Settings → Pages → Custom domain**, enter `www.codesignexplore.com`
2. At your domain registrar, add DNS records GitHub shows (usually `CNAME` for `www` → `innovis-sfu.github.io`)
3. Cancel Wix hosting after DNS propagates (24–48 hours)

## Contact form

The contact page uses [FormSubmit](https://formsubmit.co) so it works on static GitHub Pages. Messages go to `sheelagh@sfu.ca` (InnoVis lab contact).

To use a different inbox, edit `CONTACT_EMAIL` and `CONTACT_FORM_REDIRECT` at the top of `scripts/build_site.py`, then run `python scripts/build_site.py`.

**First submission:** FormSubmit sends a one-time activation link to that email address. Click it to enable the form.

## Updating content

Edit project data in `scripts/build_site.py`, then regenerate:

```powershell
python scripts/build_site.py
```

For CSS/layout changes, edit `assets/css/styles.css` directly (HTML is generated by the build script).
